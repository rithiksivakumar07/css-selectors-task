# css-selectors-task

This project demonstrates the use of different CSS selectors without using inline CSS.

## 🔹 Concepts Used

- Element Selector (p)
- Class Selector (.head)
- ID Selector (#bglime, #para)
- Descendant Selector (#bglime span, p span)
- Pseudo-class Selector (a:hover)
- Structural Selectors (:first-child, :nth-child)

## 🔹 Features

- Coral background for the entire page
- Multiple headings styled using class selector
- Second heading with lime background
- Only the word "head" in h2 is styled in red
- Paragraphs styled using element and ID selectors
- Word "paragraph" styled separately using span
- Links styled without using class or ID
- First "here" button has green background
- Second "here" button has red background
- Hover effects applied to links

## 🔹 Technologies Used

- HTML5
- CSS3

## 🔹 How to Run

1. Download or clone the repository
2. Open `index.html` in any browser
